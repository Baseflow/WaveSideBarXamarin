# WaveSideBarXamarin

Xamarin bindings library for WaveSideBar [library](https://github.com/gjiazhe/WaveSideBar).

Plugin is available on [Nuget](https://www.nuget.org/packages/Xam.Plugins.Android.WaveSideBar/).
